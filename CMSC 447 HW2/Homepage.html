Create User

By pressing the included button, you can create a user for the database.

The "GO" buttom makes an SQL statement of the format:

INSERT INTO gradebook (StudentName, StudentID, StudentScore)

Please enter a full name for the name box, and the proper numbers into the provided number boxes there is currently no update function, so make sure your entries are correct.







Search User

By filling out the field, and pressing the required button you search the database for the specified player.

The "GO" buttom makes an SQL statement of the format:

SELECT * FROM gradebook WHERE StudentName = "Your_Input"

To search for a student's information please enter the full name as it appears in the system. This will return all information about the student.




Delete User

By filling out the field, and pressing the following button you may delete a specified user from the database.

The "GO" button makes an SQL statement of the format:

DELETE FROM gradebook WHERE StudentName = "Your_Input"

Please enter a full student name that exists in the system. This will permanently delete the student from the system. This action cannot be undone.
